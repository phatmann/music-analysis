## Music Search REST Server

### Setup

#### Linux

```bash
sudo apt install ffmpeg   # for mp3 reading
```

#### Mac OS

```bash
brew install python@3.8
pip3 install virtualenv
```

#### Install

```bash
virtualenv venv3
source venv3/bin/activate
pip install -e .
```

### Run the server

```bash
cd app
./run.sh
```
